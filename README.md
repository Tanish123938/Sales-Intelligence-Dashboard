# Sales-Intelligence-Dashboard

## Project Overview

This project focuses on data cleaning, analysis, and visualization using SQL and Microsoft Power BI based on the AdventureWorks dataset.
The goal of this project is to transform raw sales data into meaningful business insights by:

- Cleaning and transforming data using SQL
- Performing data analysis
- - Building an interactive dashboard using Power BI

## Project Workflow
1️⃣ Data Cleaning using SQL
- Removed unwanted columns
- Filtered relevant data
- Handled missing values
- Formatted date columns
- Exported cleaned data into CSV files

2️⃣ Data Transformation in Power BI
- Imported CSV files into Power BI
- Established relationships between tables
- Changed data types
- Renamed columns for better understanding
- Created calculated measures

3️⃣ Dashboard Creation
- Developed a 3-page interactive dashboard:
- Sales Overview Dashboard
- Customer Analysis Dashboard
- Product Performance Dashboard

## Key Insights Generated
- Overall Sales Performance
- Customer Purchasing Behavior
- Top Performing Products
- Monthly Sales Trends
- Sales Budget vs Actual Comparison

## Conclusion

This project showcases how raw sales data can be cleaned, analyzed, and visualized using SQL and Power BI to generate meaningful business insights. The interactive dashboard helps in understanding sales trends, customer behavior, and product performance, enabling better data-driven decision-making.
